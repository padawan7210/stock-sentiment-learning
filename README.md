# stock-sentiment-learning
Learning Git, Python, and project structure using a simple stock sentiment example

## Project Structure

- data/
  Stores raw and processed data files

- scripts/
  Contains small, single-purpose scripts:
  - fetch_prices.py: retrieves stock price data
  - fetch_text.py: retrieves related text
  - analyze_sentiment.py: evaluates sentiment

- main.py
  Entry point that coordinates the workflow
